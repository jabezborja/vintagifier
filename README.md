# Installation

Prerequisites:
- Python 3.x
- Pillow Module

Assuming you already installed Python 3, you can now install Vintagifier dependencies through PIP by:
```
py -m pip install Pillow
```

After the installation, you can now run Vintagifier by:
```
py main.py
```
